## Option A: Ransmeier-Inspired

Reference read:
- Austere industrial-design minimalism.
- Black and white palette.
- Helvetica-like neutral typography.
- Full-width project structure.
- Project text can act as the main interaction trigger.
- Stacked gallery concept, where image planes accumulate from a selected project.

Frame set:
- `Option A / Ransmeier-inspired / Desktop`
- `Option A / Ransmeier-inspired / Mobile`

Visual system:
- Background: `#FFFFFF`
- Ink: `#000000`
- Soft image grey: `#DADAD6`
- Secondary grey: `#80807A`
- Typeface: Inter as a Helvetica-like substitute.
- Image treatment: stacked grey rectangles with black 1px strokes.
- Layout: generous whitespace, hard horizontal rules, sparse metadata.

Desktop composition:
- Header: Joseph York-Westcott left, Work / Studio / Contact right.
- Intro: concise positioning text on left.
- Hero: large Inter statement on right.
- Featured project: text block on left, stacked image planes on right.
- Work list: thin horizontal rows with number, title, discipline, year/status.

Suggested hero copy:
`Visual communication designer working through identity, objects, campaigns, and printed systems.`

Suggested statement:
`Images as evidence. Systems as atmosphere. Work as a sequence of decisions.`

Interaction idea:
- Hovering project titles triggers stacked image sets.
- Page stays visually still, with motion limited to image layering and title opacity.

## Option B: Public-Domain-Inspired

Reference read:
- Raw archive / OS interface.
- Command-line language.
- File/finder metaphors.
- Loud metadata and archive counts.
- Cultural, intentionally unpolished energy.

Frame set:
- `Option B / Public-Domain-inspired / Desktop`
- `Option B / Public-Domain-inspired / Mobile`

Visual system:
- Background: `#0D120E`
- Terminal green: `#B1FF4F`
- Amber highlight: `#FFA841`
- White: `#FFFFFF`
- Paper window: `#F5F2E8`
- Placeholder grey: `#BABAAE`
- Typeface: IBM Plex Mono.
- Image treatment: grey scan blocks with terminal-green outlines and scanline rules.
- Layout: terminal boot block, archive table, file-window panels.

Desktop composition:
- Top boot sequence:
  `WELCOME TO VISUAL COMMUNICATION OS...`
  `LOADING...`
  `apt-get install image-language`
  `cd $/PORTFOLIO-INDEX.sh !!!`
- Hero: `ARCHIVE OF SIGNALS: IDENTITY x IMAGE SYSTEMS`
- CTAs: `[ VIEW_WORK ]` and `[ EMAIL_JOSEPH ]`
- Archive table with project names and counts.
- Finder-style panel listing placeholder design files.

Suggested archive rows:
- `010 ARCHIVE OF SIGNALS 42`
- `009 FIELD NOTES 24`
- `008 COMMON ROOM 102`
- `007 MOVING STILL 63`
- `006 MATERIAL INDEX 13`
- `005 SOFT ARCHITECTURE 136`
- `004 AFTER IMAGE 93`

Interaction idea:
- Cursor-like blinking CTA.
- Archive rows open finder-style project windows.
- Counts animate as if loading from an index.

## Recommendation

Use the current Consul-inspired route if Joseph wants restrained credibility.

Use Option A if the portfolio should feel like an industrial/product design archive: refined, quiet, gallery-like.

Use Option B if Joseph wants a sharper cultural voice: raw, memorable, more experimental, less conventional.
